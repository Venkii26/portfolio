MY SKILLS

DEVELOPER

A developer is someone who designs, creates, tests, and maintains software or digital systems. They use programming languages and tools to turn ideas and requirements into functioning software solutions.

CODER

A coder (also called a programmer or developer) is a person who creates, tests, and maintains computer software by writing code in programming languages such as Python, JavaScript, C++, Java, etc.
